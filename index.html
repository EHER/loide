Loide

Deploy your Github Project using apt-get.

View on GitHub

Register

Log in with your Github account and enable the Loide Service.

Install

Add our Debian repository to your sources.list:

deb http://loide.eher.com.br/debian main

Update the local package index:

$ sudo apt-get update

Finally, install your project:

$ sudo apt-get install <vendor_name>-<project_name>
